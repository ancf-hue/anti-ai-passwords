🔐 Анти ИИ Пароли
Безопасный локальный генератор паролей — создавайте криптостойкие пароли без интернета, без утечек, без компромиссов.
🌟 О проекте
Анти ИИ Пароли — это социальный бесплатный проект для создания надёжных паролей прямо в браузере. Никаких серверов, никакой телеметрии, никакой передачи данных. Всё работает локально на вашем устройстве.
🔐 Принципы безопасности
Принцип
Реализация
0% утечек в сеть
Нет fetch, XMLHttpRequest, WebSocket — проект полностью офлайн
AES-256 шифрование
Сохранённые пароли шифруются через Web Crypto API перед записью в localStorage
Виртуальная клавиатура
Ввод символов через экранные кнопки — защита от системных кейлоггеров
Спецсимволы в пуле
Поддерживаются `.!@#$%^&*()_+-=[]{}
Автоочистка буфера
Пароль автоматически удаляется из буфера обмена через 30 секунд после копирования
✨ Возможности
🎯 Генерация паролей
Ручной режим: нажимайте на кружки и выбирайте символы
🎲 Случайное заполнение: одна кнопка заполняет все кружки криптостойкими символами
Длина 8–32 символа: гибкая настройка под любые требования
Drag & Drop: перетаскивайте символы для изменения порядка
🛡️ Безопасность
Индикатор надёжности: визуальная оценка силы пароля (🔴 слабый / 🟡 средний / 🟢 сильный)
Валидация ввода: разрешены только безопасные символы из белого списка
Защита от XSS: пользовательский ввод вставляется через textContent, а не innerHTML
💾 Управление паролями
Локальное сохранение: пароли хранятся в localStorage вашего браузера
Шифрование: каждый пароль шифруется перед сохранением (ключ хранится только в коде)
Экспорт/Импорт: резервное копирование в .json для переноса между устройствами
Удаление: подтверждение перед удалением, чтобы избежать случайной потери
🎨 Интерфейс
🌙 Тёмная тема: переключатель с сохранением выбора
📱 Адаптивный дизайн: оптимизирован под мобильные устройства (формат 9:19.5)
♿ Доступность: ARIA-атрибуты, поддержка клавиатуры, контрастные цвета
📳 Тактильный отклик: вибрация при нажатии (на поддерживаемых устройствах)

📖 Как пользоваться
Откройте приложение в браузере (онлайн или офлайн)
Пройдите заставку (5 секунд: картинка → видео)
Нажмите «Как пользоваться» для краткой инструкции
Перейдите в генератор и начните создавать пароль:
Нажмите на чёрный кружок → он станет красным
Выберите символ на виртуальной клавиатуре
Повторите для нужного количества кружков
Нажмите «🔐 СОЗДАТЬ ПАРОЛЬ»
Перетащите красные шары для изменения порядка символов
Скопируйте пароль (он автоматически удалится из буфера через 30 секунд)
Или сохраните — пароль зашифруется и добавится в список
🔐 Важно: Мастер-ключ шифрования хранится в коде приложения. Для максимальной безопасности в будущих версиях планируется ввод пользовательского мастер-пароля.
